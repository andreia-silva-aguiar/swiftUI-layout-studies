SwiftUI Layout Studies
This repository is dedicated to my journey in learning SwiftUI and mastering interface layouts for iOS development. It contains practical exercises and experiments with stacks, views, and modifiers.

🚀 Learning Goals
The main objective of this project is to understand the core principles of SwiftUI layout, transitioning from a Storyboard-based mindset to a Code-First approach.

🛠️ Key Concepts Explored
Stacks: Using VStack, HStack, and ZStack to structure elements.

View Modifiers: Customizing appearance through code (font, padding, colors, etc.).

Xcode Canvas: Mastering the Preview tools (Live vs. Selectable modes).

Git Workflow: Managing professional version control via Terminal.

💡 Key Takeaway
During these studies, I discovered that modern SwiftUI development in Xcode 15+ prioritizes writing code over "Drag and Drop" interactions. I've focused on using the Attributes Inspector and direct code manipulation to achieve pixel-perfect layouts, which is the current industry standard.
